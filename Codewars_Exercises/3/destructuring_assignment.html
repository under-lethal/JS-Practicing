Destructuring Assignment

Arrays

Take values from arrays or properties from objects and set them as LOCAL VARIABLES.

		New Way
		const [one, two] = [1,2,3]; 

		Old Way
		const arr = [1,2,3];
		const one = 1;
		const two= 2;

1.If you have an array - and you want to assign (some of its values to LOCAL VARIABLES), one option is to assign each variable referencing its index.Old way

		const arr = ['bacon', 'pizza', 'fries'];
		const bacon = arr[0];
		const pizza = arr[1];
		const fries = arr[2];

With DESTRUCTURING we can do the same thing - with a single line of code.
Using [] - after "const" - you can ASSIGN variables names - to each index in the array.
The position of the variable name MATCHES - the VALUE INDEX - in the array.

		const arr = ['bacon', 'pizza', 'fries'];
		const [bacon, pizza, fries] = arr;
		c.log(pizza)

2. If you want to OMIT a variable - just - adding a comma - without a variable name.

		const arr = ['bacon', 'pizza', 'fries'];
		const[,, fries] = arr;

3. In other cases - you might want to use spread operator.

		const arr = ['bacon', 'pizza', 'fries'];
		const [bacon, ...rest] = arr;
		c.log(rest); // pizza, fries.

4. You might also want to - SET a DEFAULT value - in case the value in the array is undefined.
use the = sign - to provide that fallback value.

		const arr = [undefined, 'pizza', 'fries'];
		const [bacon = 'pork', pizza, fries] = arr;

Objects

In addition to arrays - DESTRUCTURING also applies to objects.

		const obj = {
			shroom: 'mushroom',
			banana: 'banana',
			pepper: 'pepper'
		}

		//Notice how we DUPLICATING the object's property name - as a variable - with . notation.

		const shroom = obj.shroom;
		const banana = obj.banana;
		const pepper = obj.pepper;

		//By putting {} - after "const" - YOU CAN REFFERENCE PROPERTY NAMES - which automatically becomes local variables.
		const {shroom, banana, pepper} = obj;

2. Just like ARRAYS you can use "=" to set - a default value - but in some cases you may want to use a different name - than what's provided on the object itself.
Adding a ":" after the property name allows you to rename it.
It's usefull - when dealing witn NAME COLLISIONS.

			const obj = {
				shroom: 'mushroom',
				banana: 'banana',
				pepper: 'pepper'
			}

			const {shroom : mushroom}

3. You can also use a ":" - to ACCESS - nested properties.

			const obj = {
				parent: {
					child: 'child';
				},
			}

			const {parent: {child}} = obj;
			console.log(child); //child
