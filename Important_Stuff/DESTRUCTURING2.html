DESTRUCTURING - Example 2

		The idea of DESTRUCTURING - is to take an object or an array and convert it into smaller objects, ELEMENTS or VARIABLES.
	
		//Take first TWO elements from the array.
		Normaly you do something like this:

		const a = aphabet[0];
		const b = alphabet[1];
		c.log(a); //A
		c.log(b); //B
	
	< div id='d'>< /div>
	< div id='d1'>< /div>
	< script type="text/javascript">

		const alphabet = ['A', 'B', 'C', 'D', 'E', 'F', 'G'];
		const numbers = ['1', '2', '3', '4', '5', '6'];

		//Take first TWO elements from the array using DESTRUCTURING.
		//'alphabet'  - is the ELEMET you wanna DESTRUCTURE. (you put in the right side of the equal sign).
		// you say: "Destructure" this "alphabet aray". If we have an array we have these:"[]" - AROUND the ELEMENTS we want to supply.

		//So' we have our varibles "a" and "b" that we want (out of this array). "a" is the First element in this array, "b" is the Second element.
		//"c" is the third element - and so on..
		
		//If we want to skip "b" for example: const [a,, c]
		//If we want to get the rest of the elements inside of the "alpahbet": ...rest (spread operator). const [a, , c, ...rest].
		//c.log(rest); A,C, [D,E,F,G];
		//
		
		const [a, b] = alphabet;
		document.getElementById('d').innerHTML = a;
		document.getElementById('d1').innerHTML = b;
	< /script>

2. Dealing with Functions(), and returning more than one parameter.

		The Function takes twO numbers (a,b) - and SUM them Together, then MULTIPLY.
		Then return both of THEM - in an ARRAY.

	< div id="d3">< /div>
	< script type="text/javascript">
		function sumAndMultiply(a, b){
			return[a+b, a*b];
		}
		const array = sumAndMultiply(2, 3);
		document.getElementById('d3').innerHTML = array;
	< /script>

Or using Array DESTRUCTURING

	< div id="d4">< /div>
	< div id="d5">< /div>
	< script type="text/javascript">
		function sumAndMultiply(a, b){
			return[a+b, a*b];
		}
		const [sum, multiply] = sumAndMultiply(2, 3);
		document.getElementById('d4').innerHTML = sum;
		document.getElementById('d5').innerHTML = multiply;
	< /script>

	const [sum, multiply] = sumAndMultiply(2, 3);
	"sum" is the FIRST ELEMENT, "multiply" - the second elem.
	[sum, multiply] - we want to DESTRUCTURE the return from this sumAndMultiply(2, 3); -as two different variables.(sum and multiply).

3.Set default values

		const [sum, multiply, division = 'No divison'] = sumAndMultiply(2, 3);

		*by default we can SET the disivion = "No division"
		c.log(division); 5, 6, No division

		But, if we do that:
		return[a+b, a*b, a/b];
		c.log(division); 5, 6, 0.6666666

		*If the THIRD param, it's not there => 'No division'. (it gets populated to the DEFAULT value).
