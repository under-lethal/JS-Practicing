reduce()

reduce() - you'll take all the elements from an array - and returns A single VALUE.

	"This Function()" TAKES two PARAMETERS and - iterate on every element from an "arr".
	acc - accumulate the values from each item in the "arr".
	curr - are every element/items/values from the "arr".

< div id="r">< /div>
< div id="r1">< /div>
< script type="text/javascript">
	// sum of all elements
	
	function findSum(arr){
		let sum = 0;
		for(let i = 0; i < arr.length; i++){
			sum = sum + arr[i];
		}
		return sum;
	}
	document.getElementById('r').innerHTML = findSum(arr);	

	// do the same thing - using reduce();
	const output5 = arr.reduce((acc, curr) => {
		acc += curr; 
		return acc;
	} ,0);
	document.getElementById('r1').innerHTML = output5;

Find MAX - using reduce()

	1. Itterate the array.
	2. Find if the arr[i]/CURRENT value/ any of thses array values[...] is  >  than maximum.
	3. Overwrite our maximum with arr[i]. max BECOMES arr[i].
	4. return max.

< div id="mx">< /div>
< script type="text/javascript">
	//find the MXIMUM in this array:
	function findMax(arr){
		let max = 0;
		for(let i = 0; i < arr.length; i++){
			if(arr[i] > max){
				max = arr[i]; --- overwrite
			}			
		}
		return max;		
	}
	document.getElementById('mx').innerHTML = findMax(arr);		


	Use the same LOGIC - using reduce();
	1. Use method, that takes a Function - with two param: (acc, curr).
	2. } ,0); - the SECOND ARGUMENT of this Function() - would be - the intial value of accumulator. // let max = 0;
	3. acc - acting like "max". For each value/elem (from the arr) - we will CHECK - if (curr > acc)
	acc - is maximum value.
	4. ASSIGN the MAX VALUE(curr) to that "acc".

	//using reduce();
	const output6 = arr.reduce((acc, curr) => {
		if(curr > acc){
			acc = curr;
		}
		return acc;
	} ,0);

reduce() - on Array of Objects

	1. Use reduce() Func. - with two PARAMETERS.
	1.1 We're using this array - and we need to FIND one VALUE - inside of it. We don't want a list.
	1.2 We want an OBJECT, and object should have different unique VALUES (with the count of individual values).
	{26: 2, 75 : 1, 50: 1}
	We want to reduce our ARRAY - to a PARTICULAR VALUE - which is an OBJECT.
	
	2. The second ARGUMENT -(to this reduce Function)- will be - The initial value - of this "acc". 
	acc = {26: 2, 75 : 1, 50: 1}
	An EMPY OBJECT. {}
	We will slowly be APPENDING (adauga/anexeaza) these values to IT. acc = {26: 2, 75 : 1, 50: 1} 
	
	3. When this Function RUNS/iterate against each of these elements of this array.
	curr - will be - the ENTIRE/this particular object from the ARRAY:
	{firstName: 'akshay', lastName:'saini', age: 26},

	4. Check if(acc[curr.age]) - check if acc of curr.age.
	4.1 Check if the acc of 26 it's there or not. (in that EMPTY OBJECT) {}.
	4.2 Because is an empty OBJECT the acc[curr.age] is not there.
	If it is there - we will do something - if not... we ASSING acc[curr.age] = with 1;

	5. This function will RUN for Each of Every Value (inside the array) - to check is the "acc of 26, 75 50" - is
	it or it is not there {}. If IS NOT - acc[curr.age] = 1

	6. If acc of [curr.age] - it IS ALREADY present it will simply  ++acc[curr.age]

< div id='rao'>< /div>
< script type="text/javascript">
	// how many USERS have a PARTICULAR AGE/the same age.
	// output: acc = {26: 2, 75 : 1, 50: 1}
	const output8 = users.reduce((acc, curr) => {
		if(acc[curr.age] ){
			acc[curr.age] = ++acc[curr.age];
		}else{
			acc[curr.age] = 1;
		}
		return acc; 
	}, {});
	document.getElementById('rao').innerHTML = output8;
< /script>
