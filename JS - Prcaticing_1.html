JavaScript Operators

x = 5, y = 2, calculate z = x + y, and display "z":

	< p id="demo5">< /p>
	< script>
		let x = 5;
		let y = 2;
		let z = x + y;
		document.getElementById("demo5").innerHTML = z;
	< /script>

JavaScript String Operators

The + operator concatenates (adds) strings.

	< p id="demo6">< /p>

	< script>
		let text1 = "John";
		let text2 = "Doe";
		let text3 = text1 + text2;
		document.getElementById("demo6").innerHTML = text3;
	< /script>

The assignment operator += can concatenate strings.

	< p id="demo">< /p>

	< script>
		let text1 = "What a very ";
		text1 += "nice day";
		document.getElementById("demo").innerHTML = text1;
	< /script>

What a very nice day!

Adding Strings and Numbers

Adding a number and a string, returns a string.

	< p id="demo7">< /p>

	< script>
		let x = 5;
		let y = "5" + 5;
		let z = "Hello" + 5;
		let a = 5 + "Hello";
		document.getElementById("demo7").innerHTML = x + "< br>" + y + "< br>" + z + "< br>" + a;
	< /script>

5, 55, Hello5, 5Hello

JavaScript Arithmetic

The % Operator

The modulus operator (%) returns the division remainder.

	< p id="demo8">< /p>
	< script type="text/javascript">
		let x = 5;
		let y = 2;
		let z = x % y;
		document.getElementById('demo8').innerHTML = z;
	< /script>

1

Incrementing

The increment operator (++) increments numbers.

	< p id="demo9">< /p>

	< script>
		let x = 5;
		x++;
		let z = x;
		document.getElementById("demo9").innerHTML = z;
	< /script>

6

Decrementing

The decrement operator (--) decrements numbers.

	< p id="demo 10">

	< script type="text/javascript">
		let x = 5;
		x--;
		let z = x;
		document.getElementById('demo 10').innerHTML = z;
	< /script>

5

JavaScript Assignment Operators

The += assignment operator adds a value to a variable.

	< p id="demo">< /p>

	< script>
		let x = 10;
		x += 5;
		document.getElementById("demo").innerHTML = x;
	< /script>

15

	The -= assignment operator subtracts a value from a variable.
	Assignment
	let x = 10;
	x -= 5;

	The *= assignment operator multiplies a variable.
	Assignment
	let x = 10;
	x *= 5;

	The /= assignment divides a variable.
	Assignment
	let x = 10;
	x /= 5;

JavaScript Data Types

JavaScript variables can hold different data types: numbers, strings, objects and more:

	let length = 16;                               // Number
	let lastName = "Johnson";                      // String
	let x = {firstName:"John", lastName:"Doe"};    // Object

JS Types are Dynamic

JavaScript has dynamic types. This means that the same variable can be used to hold different data types:

	let x;           // Now x is undefined
	x = 5;           // Now x is a Number
	x = "John";      // Now x is a String
