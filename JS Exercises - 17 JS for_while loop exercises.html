";
		}
		
	}

	we use the variable "j" (instead of "i"), because we don't want to keep track the iteration of this ^| loop.
	*On the first Iteration of the "i" loop - we actually loop throught the entirety of "j".
	So - "j" is loop through all the way (for one interation) of the "i" loop.
	0, 0, 0, - is all on the same loop number of "i".
	"j" loop through 3 times. (j=0, j=1, j=2);

	So - For Every iteration of "i" - "j" will fully run/ we gonna get through every iteration of 'j' loop. (the loop inside of "j" it's gonna get FULLY EXECUTED).

Let's make a different way - of doing this: (the other/better way - is using "splice"
	'Splice' are sounds very simmilar to "slice" - but they are very different when acting on arrays.
	"Splice" allows you to ADD and to REMOVE elements to an Array. It does Both things.

	Let's remove that 3'rd element - with 'splice()'

	let arry =  [1,2,3,4,5];
	arry.splice(2, 1); //2 - indicate the index, '1' - how many elements I want to remove.
	What this will return - is the element/elements that we're remove.
	As we can see "splice()" mutate/changes the array you're working on.
If you want to remove an 'element' but not to CHANGE/mutate the original Array - you could CLONE the array and remove the elements.

In order to do that - we neet to SET UP a Function()

< p id="rem">< /p>
< p id="rem1">< /p>
< script type="text/javascript">
	let arry = [1,2,3,4,5];
	let output = '';
	let output1 = '';

	const removeElements = function(a, index){
		let newArray1 = [...a];
		newArray1.splice(index, 1);
		return newArray1;
	};
	let newArray1 = removeElements(arry, 2);
	document.getElementById('rem').innerHTML = output += arry;
	document.getElementById('rem1').innerHTML = output1 += newArray1;
< /script>

	1. Call a function (removeElements) - pass an ARRAY and 'index of what we want to remove'. (function(a, index)).
	2. In order to "mutate" this array - even it's passed into a NEW VARIABLE. function(a, index) - passed by refference in JS. [if we will change it (with'a') - it will mutate the array]. 'a' is 'arry'.
	3. IN ORDER TO PREVENT THAT FROM HAPPENING  - we first need to CLONE IT.
	3.1 A really quick way to CLONE an Array is using a "SPREAD OPERATOR" - [...a]
	3.2 The "spread OP" - will take this array 'a'/arry  - and it SPREADS OUT all the Elements.
	...and it spreads them all out - inside of this square brackets '[...a]' - ASSIGNED TO "let newArray' VARIABLE.
	4. Once we get that "newArray" - then we can do the splice. newArray.splice(index, 1)
	5. Finally - return newArray.
	that ALLOWS us to modify the array - without MUTATING  the original array.
	6. It will give us a NEW Array in return.
	let newArray = removeElement(arry, 2)
	2 - in this case  - indicates the INDEX THAT WE WANT TO REMOVE.
