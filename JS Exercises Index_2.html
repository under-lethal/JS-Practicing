The for...of loop

The "for...of loop" - is similar to the "for loop", but it will loop over each item in the array...

The 'for...of' LOOP throught property VALUES.
"for of" - loops through ITERABLE(REPREZENTARE a unei serii de elemente care pot fi REPETATE) OBJECTS. (array, maps, set...).
"for..in" - loops through INUMERABLE (care se pot NUMARA) OBJECTS.
for of - will not work on a "person object" - because person is not ITERABLE.

"for...in" in an Array

for of - will not work on a "person object" - because person is not ITERABLE.
The "for...in" - loops through INNUMERABE (care nu se pot numara) PROPERTIES - while "for...of" loops through ITERABLE (*reprezentare a unor serii de elemente care pot fi repetate) OBJECTS such as (arrays, sets, map).

The for ... in loop is very similar to the for ... of loop. The only difference is that instead of giving you the item in the array, it gives you the index each time through the loop.
As you can see, we don't need to use index notation to get the item in the array; it's just given to us each time through the array.
When you use a for ... of loop, you get access to each element in the array, one each time through the loop. When you use a for ... in loop on an array, however, you get access to the index of each item in the array.

for...each

The method takes a function (as the argument), and loops over each item in the array:

For every Element from (an array) - we're gonna call a Function() (one time for each element from the array) that i put inside of forEach().

	< p id="fe">< /p>
< script type="text/javascript">

	let newNumb = '';
	numb.forEach(function(item){return item});

< /script>
