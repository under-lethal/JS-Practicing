JS-Objects

JavaScript Objects

In JavaScript almost everything is an object.

  • Booleans can be objects (if defined with the new keyword)
  • Numbers can be objects (if defined with the new keyword)
  • Strings can be objects (if defined with the new keyword)
  • Dates are always object
  • Maths are always objects
  • Regular Expressions (RegEx) are always objects
  • Arrays
  • Functions
  • Objects


1.Creating a JavaScript Object

Using an "Object Literal"

		const person = {
			fName: 'John',
			lName:'Doe',
			age: 50,
			eyeColor: 'blue'
		};

		document.getElementById('ol').innerHTML = person.fName + " is " + person.age + " years old." ;

Using the JavaScript Keyword new

		const personObj = new Object();
		personObj.fName = 'Axl';
		personObj.lName = 'Rosses';
		personObj.age = 50;
		personObj.eyeColor = 'blue';

		document.getElementById('obj').innerHTML = personObj.fName + " is " + personObj.age + " years old." ;


There is no need to use new Object().

For readability, simplicity and execution speed, use the object literal method.


2.Object Properties

Accessing JavaScript Properties

The syntax for accessing the property of an object is:

	objectName.property // person.age
	objectName.['property'] // person["age"]
	objectName.[expression] //  x = 'age'; person[x]

JavaScript for...in Loop

The JavaScript for...in statement loops through the properties of an object.

	const personL = {
		fname:" John",
		lname:" Doe",
		age: 25
	};
	
	let txt = '';
	for (let x in personL){
		txt += personL[x] + ' ';
	}
		document.getElementById("fi").innerHTML = txt;

Adding New Properties

Example:

	person.nationality = 'English';

Deleting Properties

	const person = {
		fName: 'Axl',
		lName: 'Rosses',
		age: 50,
		eyeColor: 'blue'
	};

	delete person.age;
	delete person['age'];


	console.log(person.fName + ' is ' + person.age + ' years old.');
	
	Output:
	John is undefined years old.

Nested Objects

Values in an object can be another object:

	const myObj = {
		name: 'John',
		age: 30,
		cars: {
			car1: 'Ford',
			car2: 'BMW',
			car3: 'Fiat'
		}
	};

	document.getElementById("demo").innerHTML = myObj.cars.car2;
	document.getElementById("demo").innerHTML = myObj.cars['car2'];
	document.getElementById("demo").innerHTML = myObj.['cars'].['car2'];

Nested Arrays and Objects

Values in objects can be arrays, and values in arrays can be objects:

	const myObj = {
		name: 'John',
		age: 30,
		cars [
			{name: Ford, models:["Fiesta", "Focus", "Mustang"]},
			{name: BMW, models:["320", "X3", "X5"]},
			{name: Fiat, models:["Multipla", "500", "Panda"]},
			]
	};

To access arrays inside arrays - use for...in loop.

	for (let i in myObj.cars){
		x += '< h2>' + myObj.cars[i].name + '< h2>';
			for(let j in myObj.cars[i].models){
				x += myObj.cars[i].models[j] + '< br>'
			}
	}

	document.getElementById("demo").innerHTML = x;

JavaScript Accessors (Getters and Setters)

JavaScript Getter(The get Keyword)

Create an object. | Get the value of the 'language' property.

	const getPerson = {
		firstName: 'Joe',
		lastName: 'Peterson',
		language: 'en',
		get lang() { 	//get the value of language property.
			return this.language;
		}
	};
	
	document.getElementById('get').innerHTML = getPerson.lang;

JavaScript Setter(The set Keyword)

	const setPerson = {
		name: 'Ana',
		surName: 'Pauker',
		language:'',
		set lang(lang){
			this.language = lang;
		}
	};

	setPerson.lang = 'german';
	document.getElementById('set').innerHTML = setPerson.language;

The differences between Function & Getter:

	Function access fullName as a function: person.fullName().

	Getter access fullName as a property: person.fullName

	JavaScript CAN SECURE better Data Quality when using Getters and Setters.

JavaScript Object Constructors

Sometimes we need a "blueprint" for creating many objects of the same "type.
Objects of the same type - are created by calling the constructor function with the new keyword.

	function Person(first, last, age, eye){
		this.fName = first;
		this.lName = last;
		this.age = age;
		this.eyeColor = eye;
	}

	const myFather = new Person ('John', 'Doe', 50, 'blue');
	const myMother = new Person ('Sally', 'Rally', 45, 'green');

	document.getElementById('cnstr').innerHTML = 'My father is ' + myFather.age + '. My mother is ' + myMother.age + '.';

Adding a Property to an OBJECT

	myFather.nationality = 'German';

	// Constructor function for Person objects
	function Person(first, last, age, eye) {
		this.firstName = first;
		this.lastName = last;
		this.age = age;
		this.eyeColor = eye;
	}

	// Create 2 Person objects
	const myFather = new Person("John", "Doe", 50, "blue");
	const myMother = new Person("Sally", "Rally", 48, "green");

	// Add nationality to first object
	myFather.nationality = 'German';

	// Display nationality 
	document.getElementById("demo").innerHTML =
	"My father is " + myFather.nationality;

Adding a Property to a CONSTRUCTOR

	function Person(first, last, age, eyecolor) {

		this.age = age;
		this.eyeColor = eyecolor;

	document.getElementById("demo").innerHTML =

Adding a Method to an OBJECT

	// Constructor function for Person objects
	function Person(first, last, age, eye) {
		this.firstName = first;
		this.lastName = last;
		this.age = age;
		this.eyeColor = eye;
	}

	// Create 2 Person objects
	const myFather = new Person("John", "Doe", 50, "blue");

		return this.firstName + " " + this.lastName;
	};

	// Display full name
	document.getElementById("demo").innerHTML =
	"My father is " + myFather.name();
